## Overview

You can find various resources to learn about different wallet integrations in the Internet Computer and how to integrate them into your applications.

## Resources

- [ic-auth](https://github.com/id-daniel-mccoy/ic-auth) - A set of tooling designed to make it easy for Internet Computer developers to integrate IC authentication into their apps.
- [Official Docs](https://internetcomputer.org/docs/current/developer-docs/defi/wallets/overview) - Official documentation for wallets in the Internet Computer.


<Note title="WARNING" type="warning">
We do not recommend using **ic-auth** in production as it is still in the early stages of development. 
</Note>
